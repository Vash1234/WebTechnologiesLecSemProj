HTML

What is HTML?

  • Used to create and mark up web pages
    • Markup Language - used to get resource and identify sections of the resource
  • Used for Structure and Content of Web Development
  • Invented by Sir Tim Berners-Lee
  • World Wide Web Consortium (W3C) oversees the development of HTML

Versions

  • HTML 1.0
    • Very limited, no formal specifications
  • HTML 2.0
    • RFC 1866
    • Evolution of HTML
  • HTML 3.0
    • Immediately super seeded by 3.2 -> 4.0 (1997) -> 4.01 (1999)
    • 4.01 - XHTML (1.0, 1.1, 2.0)
      • XML - data markup, data interchange, very strict
      • Strict version, transitional version, frameset version
      • Hickson - WHATWG
        • on evolution of HTML 4.01 -> HTML5 (2014)
